osd_cl_mam class
----------------

A client for the Memory Access Module (MAM).

``cl_mam`` is the counterpart to the Memory Access Module (MAM): it implements the MAM protocol to read and write data from and to memories.

Usage
^^^^^

.. code-block:: c

  #include <osd/osd.h>
  #include <osd/cl_mam.h>
